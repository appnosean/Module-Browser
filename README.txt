README.txt
==========

A module that provides an easier interface to the new installation method.
This by providing a list of available modules

INSTALLATION
============
Install as usual, see http://drupal.org/node/70151 for further information.

USAGE
=====
Browse to 'admin/modulebrowser', there you can choose between modules and themes to install.
Find the module/theme to install, click the checkbox and provide login data for server.
The module will be downloaded, but not yet enabled. 
Enabling the modules will become an extra step in the installation process.
